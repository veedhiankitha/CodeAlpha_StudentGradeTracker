# Student Grade Tracker

A Java console application developed for the CodeAlpha Java Internship.

## Features
- Add student names and marks
- Calculate average marks
- Display highest marks
- Display lowest marks
- Generate a student summary report

## Technologies Used
- Java
- VS Code
## ▶️ How to Run

1. Open the project in VS Code.
2. Compile the program:

bash
javac StudentGradeTracker.java


3. Run the program:

bash
java StudentGradeTracker


4. Enter the student details when prompted.
